# Sankeerth Bussa — Portfolio

Modern portfolio site built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

## Features
- 🌑 Dark theme with glassmorphism design
- ✨ Scroll-triggered animations (Framer Motion)
- 📱 Fully responsive (mobile-first)
- ⚡ Static export ready (GitHub Pages / Vercel / any CDN)
- 🎨 Gradient text, floating orbs, grid background

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

```bash
npm run build
# Static output in /out — deploy anywhere
```

## Tech Stack
- Next.js 14 (App Router, static export)
- Tailwind CSS 3
- Framer Motion 11
